《大明浮生志3》隐私政策

更新日期：2026年7月21日　生效日期：2026年7月21日

欢迎使用《大明浮生志3》（以下简称“本应用”）。我们重视您的个人信息与隐私安全。本政策说明本应用如何处理个人信息、使用第三方 SDK，以及您如何行使相关权利。请在使用前仔细阅读。

一、我们处理的信息

信息类型使用目的处理方式
TapTap 账号公开资料、OpenID/UnionID账号登录、识别登录状态、实名认证与防沉迷由 TapTap SDK 依照其规则处理；本应用仅使用登录结果和用户标识
设备型号、操作系统版本、网络状态服务适配、广告请求、故障诊断仅在您同意本政策后由相关 SDK 处理
Android ID、OAID广告展示、归因统计、频次控制及反作弊由广告 SDK 在授权能力范围内处理
广告展示、点击、完成及奖励校验记录完成激励广告流程并发放游戏内奖励通过聚合广告 SDK 处理
游戏存档、声音及字体设置保存游戏进度与偏好默认仅存储在您的本地设备

二、设备权限

本应用仅申请提供账号登录和广告服务所需的网络访问、网络状态与 Wi-Fi 状态权限。我们通过广告 SDK 的隐私控制关闭电话状态、MAC 地址、精准定位、录音和已安装应用列表等非必要能力，不主动申请通讯录、短信、相机或麦克风权限。

SDK用途可能处理的信息

第三方 SDK 仅在您明确同意本政策后初始化。第三方处理活动还受对应服务提供者隐私政策约束。

四、信息共享、转让与公开披露

除实现上述登录、合规和广告服务所必需，或法律法规另有要求外，我们不会出售您的个人信息，不会擅自向无关第三方共享、转让或公开披露您的个人信息。

五、保存期限与安全

我们仅在实现本政策所述目的所需的最短期限内保留相关信息。游戏存档默认保存在本机，卸载应用或清除应用数据可能导致存档删除。我们采用访问控制、最小权限和 SDK 延迟初始化等措施降低信息泄露、滥用或丢失风险。

六、您的权利

您有权依法查询、更正、复制、删除个人信息，撤回同意或注销账号。您可以选择“不同意并退出”；同意后也可通过系统设置清除本应用数据以撤回本地同意记录。账号资料及注销请求可通过 TapTap 提供的账号管理渠道办理。撤回同意不影响撤回前基于同意进行处理的合法性。

七、未成年人保护

本应用接入 TapTap 实名认证和防沉迷服务，并依据认证结果执行可玩时段和时长限制。未成年人应在监护人指导下阅读和使用本应用。

八、政策更新

当处理目的、处理方式或第三方 SDK 发生重大变化时，我们会更新本政策，并通过应用内弹窗或发布页面重新提示。未经重新同意，不会将个人信息用于与原目的无直接或合理关联的新用途。

九、联系我们

如对本政策或个人信息处理有疑问、投诉或权利请求，请通过本应用在 TapTap 商店开发者页面公示的联系渠道与我们联系。我们将在核验身份后依法处理。
