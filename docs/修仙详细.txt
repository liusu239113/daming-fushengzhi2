一、游戏整体定位底层设计逻辑
————————————————————————————
 
1. 核心定位设计初衷
设计初衷：做一款低操作、高沉浸、慢节奏宗族挂机模拟器，不靠打斗爽感，靠「血脉传承+家族兴衰」给玩家情绪代入。
设计思路：放弃传统闯关、副本、即时战斗，把「族谱树」当做唯一核心载体，让玩家视觉上直观看到家族一代一代繁衍、生老病死、兴衰变化。
2. 玩法基调设计
刻意设计成：离线可挂机、时间自动流逝、不用高频操作、碎片化随时上线打理，适配手游佛系玩家。
3. 核心底层逻辑设计
整套游戏所有模块，全部围绕一条主线闭环设计：
族人出生→成长修炼→婚配生子→扩充家族→资源经营→应对随机事件→世代延续→家族升级变强。
所有界面、所有系统、所有功能，都是为这条闭环服务，没有多余无关玩法。

二、主界面每一块布局 设计原理+为什么这么排布
————————————————————————————
 
1. 顶部固定信息栏 设计逻辑
设计初衷：把玩家最关心的「家族等级+核心资源」放在第一眼视线区，不用翻菜单就能看到关键数据。
布局设计原因：
左侧放家族名号和品级，强化玩家「自己创建家族」的归属感；
中间放灵石仙晶两大核心资源，是整个游戏的流通命脉，必须常驻可视；
右侧放设置、日志、公告，属于低频常用功能，放在角落不干扰主视觉。
2. 中间全屏族谱树 设计逻辑
设计初衷：这是游戏灵魂核心，刻意把最大屏幕区域留给族谱，让玩家视觉聚焦族人血脉关系。
结构设计规则：
刻意做成「长辈在上、晚辈在下、同辈横向排列」，符合中国人宗族族谱传统认知，降低理解成本；
夫妻连线、子嗣分支可视化，不用看文字，一眼看懂家族关系；
支持拖动+缩放，是为了后期家族几十上百人时，能自由查看各分支世代。
节点设计逻辑：每个族人只放头像+境界+小状态标识，极简信息，避免界面杂乱。
3. 左侧缩放控制栏 设计逻辑
设计初衷：专门服务族谱视图，给手残玩家快捷放大缩小，不用双指操作，降低上手门槛。
4. 右侧竖排快捷功能栏 设计逻辑
设计初衷：把日志、功法、产业、库房这些高频子功能，做成侧边常驻，不用每次点底部菜单，提升操作效率。
5. 底部固定导航栏（图腾/功法/产业/宝物/仙峰/库房）设计逻辑
设计初衷：拆分游戏六大核心系统，做成固定入口，结构标准化，玩家记忆成本低。
排布逻辑：从家族buff→族人修炼→资源产业→道具宝物→修炼加成→仓库存储，按玩家游玩顺序从左到右排布，符合操作习惯。
6. 右下角时间调速栏 设计逻辑
设计初衷：时间流逝是游戏核心驱动，必须常驻可视；
设计倍速、暂停、播放，是为了满足玩家两种需求：
佛系玩家正常速度慢慢看家族兴衰；
急性玩家开倍速快速跳年代、看世代更替。
7. 全局弹窗体系 设计逻辑
设计初衷：所有详情、事件、结算全部用弹窗，不跳转新页面，保持主族谱界面不被打断，沉浸感不丢失。
弹窗统一规格：固定大小、居中弹出、半透明背景遮罩，视觉统一，玩家不用适应不同界面样式。

三、开局流程 每一步设计逻辑
————————————————————————————
 
1. 主菜单设计逻辑
只保留新游戏、继续游戏、存档管理、设置四个按钮，刻意做极简设计，不让新手迷惑。
设计目的：专注开局和存档，无多余广告、活动弹窗，保持单机纯粹感。
2. 新游戏三步创建 设计逻辑
第一步选姓氏：给玩家自定义家族的代入感，有「自建宗族」的身份认同；
第二步选家族图腾：给全局固定buff，拉开每局开局差异，增加重复游玩性；
第三步选开局出身/难度：照顾新手和老玩家，新手简单开局，老玩家困难开局挑战生存。
3. 进入主界面自动生成初代老祖 设计逻辑
不用玩家手动创建人物，系统自动生成，降低开局操作门槛，直接进入游玩循环。

四、七大核心玩法模块 逐项底层设计逻辑
————————————————————————————
 
【模块一 宗族族谱&族人管理系统 设计逻辑】
 
1. 族人基础属性设计初衷
设计姓名、年龄、灵根、境界、寿元、天赋、状态，是为了给每个族人做独立人设，不是统一工具人，让玩家有培养偏爱感。
2. 族谱关系自动记录设计
系统自动绑定父母、配偶、子女关系，不用玩家手动编辑，降低操作成本，符合模拟经营懒人逻辑。
3. 族人状态分类设计
刻意拆分：健康、受伤、修炼、历练、怀孕、寿元将尽、陨落，每一种状态对应不同玩法影响，
怀孕对应子嗣系统、历练对应外出玩法、陨落对应家族兴衰情绪。
4. 婚配&子嗣系统设计初衷
核心目的：保证家族可以世代延续，是游戏能无限循环的根基；
设计族内联姻+跨家族联姻：
族内联姻保血脉纯度，跨家族联姻给资源、声望、功法，给玩家策略选择；
设计子嗣继承灵根天赋：制造随机性，每一代族人资质不一样，每局游玩体验不同；
设计过继、逐出族人：给玩家管理宗族的权力，淘汰废柴、保留优质血脉，增加策略性。
 
【模块二 灵根&天赋&境界养成系统 设计逻辑】
 
1. 灵根系统设计初衷
用灵根区分族人资质高低，制造天生差距，让玩家有择优培养的目标；
单灵根快、多灵根潜力高，设计差异化，不让玩家只盯着一种灵根玩。
2. 天赋系统设计逻辑
给每个族人随机特长（炼丹、炼器、悟性、战斗），引导玩家分工培养，有人修炼、有人炼丹、有人打架，形成家族分工体系。
3. 境界等级设计逻辑
做固定大境界+每层小境界，拉长养成线，给玩家长期追求目标；
境界绑定寿元：修为越高活越久，倒逼玩家必须培养族人升级，不然老辈快速陨落、家族断层。
4. 修炼&突破&天劫设计逻辑
自动挂机修炼：贴合佛系懒人玩家；
丹药加速：给氪金/肝玩家快速提升途径；
突破失败+天劫风险：增加不确定性，有惊喜也有损失，拉高情绪起伏；
高阶族人护法：增加策略玩法，不是无脑堆资源就能突破。
 
【模块三 功法&仙峰系统 设计逻辑】
 
1. 功法品级设计（凡/灵/地/天）
拉长收集和养成线，后期慢慢解锁高阶功法，给长期留存支点。
2. 功法适配灵根设计
属性匹配有加成，引导玩家策略搭配，不是随便给功法就行，增加玩法深度。
3. 仙峰系统设计初衷
专门做全局修炼加成，作为家族升级核心福利；
解锁修炼席位：限制同时修炼人数，倒逼玩家升级仙峰、规划族人分工，不能全员同时挂机修炼。
 
【模块四 资源&产业经营系统 设计逻辑】
 
1. 双货币设计（灵石+仙晶）逻辑
灵石日常通用、仙晶高阶稀缺，拉开资源层级，区分普通消耗和顶级消耗，控制游戏数值节奏。
2. 灵草矿石材料设计
为炼丹、炼器做前置供给，形成「产业产出材料→炼制丹药法宝→反哺族人修炼」的内循环。
3. 多产业设计（灵田/矿场/商铺/药圃）
设计目的：拆分资源产出渠道，每种产业对应一种资源，玩家需要均衡升级，不能单点发育；
指派族人驻守加成：增加族人利用策略，不让闲人闲置。
4. 每月自动结算设计
离线也能产出资源，完美适配挂机慢节奏玩法。
 
【模块五 随机事件抉择系统 设计逻辑】
 
1. 事件分类设计（家族内部/奇遇/危机/天地大事）
设计初衷：打破单调挂机，每隔一段时间给玩家新事情处理，盘活每日上线动力。
2. 多选项抉择设计
每个事件2-3个选择，不同结果影响资源、族人、声望，制造策略取舍，没有绝对正确答案，提升重复游玩性。
3. 事件记入家族日志
强化家族历史感，让玩家能回看自己家族一路走来的兴衰历程，增加情感代入。
 
【模块六 历练&回合战斗系统 设计逻辑】
 
1. 外出历练设计初衷
给资源、功法、稀有天赋额外获取渠道，不局限于产业自产；
耗时挂机设计：符合整体慢节奏，不用即时操作。
2. 多历练目的地设计
秘境/妖兽/遗迹/敌对家族，不同目的地奖励和风险不同，给玩家策略选择。
3. 回合战斗+倍速跳过设计
战斗只是附属玩法，不是核心，所以做简化回合制，支持倍速跳过，不占用玩家过多时间，只作为历练结果的载体。
 
【模块七 家族等级&声望&周目系统 设计逻辑】
 
1. 家族品级逐级升级设计
给玩家宏观长期目标：从小家族一步步做成顶级仙族，拉长长线游玩周期。
2. 声望系统设计逻辑
作为隐形稀有资源，解锁高阶联姻、降低危机概率、提升产业收益，给玩家额外养成追求。
3. 图腾系统设计
全局永久buff，每局开局选择不同图腾，玩法体验不一样，增加重玩价值。
4. 周目轮回设计逻辑
设计家族覆灭重置机制，避免玩家玩到无敌后无聊；
新周目继承部分buff，让玩家下一局开局更有优势，愿意反复开新档游玩。

五、游戏核心大循环 刻意设计的闭环逻辑
————————————————————————————
 
1. 时间自动流逝是整个游戏的核心驱动
所有养成、繁衍、资源产出、事件触发，全部绑定时间走，不用玩家手动推进，适配挂机。
2. 循环链路设计逻辑
开局造家族→生成初代族人
分配族人修炼/守产业/去历练
时间流逝→每月结算资源+族人变老
随机事件触发→玩家做选择
用资源升级产业、仙峰、功法
族人突破境界、婚配生子
族谱壮大、家族品级提升
遭遇危机处理灾祸
世代不断传承
家族覆灭→开启新周目重新开局
整套链路环环相扣，每一个系统都不是孤立，全部嵌入这条循环里。

六、数值体系 整体设计逻辑
————————————————————————————
 
1. 资源数值：低增速、慢积累，刻意控制节奏，不让玩家快速毕业，拉长游玩时间。
2. 境界数值：逐层抬高升级所需资源，倒逼玩家经营产业、培养更多族人。
3. 灵根天赋数值：随机概率分布，极品稀有、普通居多，制造养成期待感。
4. 寿元数值：低境界寿命短、高境界寿命长，强制玩家持续培养后辈，避免单一老祖躺平通关。

七、玩家留存 刻意设计的底层套路逻辑
————————————————————————————
 
1. 短期留存设计
靠每日随机事件、每月资源结算、族人状态变化，让玩家每天上线都有事情要处理，不会无事可做。
2. 中期留存设计
靠族人世代养成、族谱慢慢变大、产业逐步升级、境界一层一层突破，给玩家看得见的成长反馈。
3. 长期留存设计
靠高阶功法、满级仙峰、顶级家族品级、多世代血脉传承，设置长线终极目标。
4. 佛系留存设计
离线挂机、时间自动流逝、不用高强度操作，适配休闲玩家，挂着就能慢慢发展家族。
5. 重复游玩留存
图腾随机、开局出身随机、事件随机、族人天赋随机，每一局玩法走向都不一样，愿意反复开新档。
